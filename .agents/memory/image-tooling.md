**Rule:** Don't try to install `sharp` (npm) — the install fails here. For image work use CLI tools:
- `rsvg-convert` (installed via nix `librsvg`) — SVG → PNG rendering; fonts render fine (DejaVu family available), use English-only text in generated SVGs to avoid Arabic shaping risk.
- `magick` / `convert` and `cwebp` — already on PATH; use for resize + WebP compression (e.g. `magick in.png -resize "1600x>" -quality 80 out.webp` shrank ~1MB heroes to ~55KB).

**Why:** `installLanguagePackages` failed for sharp (2026-07); nix CLI tools worked immediately.

**How to apply:** Card artwork for the fazaa project is generated by the rerunnable script `artifacts/fazaa/scripts/generate-card-images.mjs` → `public/cards/<brand>/<type>.png`.
