تمام، احكيله هذا النص:

أريد تحويل المشروع الحالي بالكامل من Vite + React إلى Next.js باستخدام App Router وTypeScript، وليس فقط تغيير أسماء المجلدات.

المطلوب:

1. حافظ على التصميم الحالي والصفحات والوظائف كما هي تماماً.
2. استبدل التنقل المستخدم في React مثل react-router-dom بنظام Next.js:

import Link from "next/link";
import { useRouter } from "next/navigation";

3. أي صفحة أو مكوّن يستخدم useState أو useEffect أو الأحداث أو Firebase في المتصفح، أضف في بدايته:

"use client";

4. انقل الصور والشعارات والأصوات إلى public.
5. انقل إعداد Firebase والدوال المشتركة إلى lib.
6. انقل المكونات المشتركة إلى components والـ hooks إلى hooks.
7. انقل CSS العام من src/index.css إلى app/globals.css واربطه داخل app/layout.tsx.
8. احذف ملفات Vite بعد نجاح التحويل:

vite.config.ts
index.html
src/main.tsx
src/App.tsx
dist/
tsconfig.tsbuildinfo

9. احذف react-router-dom فقط بعد تحويل جميع الروابط والمسارات.
10. لا تحذف قاعدة البيانات أو متغيرات البيئة، ولا تغيّر أسماء Collections أو حقول البيانات.
11. لا تنشئ ملفات إعداد مكررة. احتفظ بملف واحد فقط لكل إعداد.
12. أصلح جميع الاستيرادات بعد نقل الملفات.
13. أضف نسخة احتياطية أو Git commit قبل التحويل.
14. بعد الانتهاء شغّل:

npm install
npm run lint
npm run build

لا تعتبر التحويل منتهياً إلا إذا نجح npm run build بدون أخطاء، وعملت جميع المسارات والصفحات وربط Firebase كما كانت في المشروع القديم.

مهم: لا تخليه فقط ينقل pages إلى app؛ لازم يعدّل نظام التنقل والكود الذي يعتمد على Vite، وإلا المشروع لن يعمل كـ Next.js.
